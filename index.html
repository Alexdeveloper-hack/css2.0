Propiedad Valor Inicial Descripción
color inherit Establece el color del texto de un elemento.
font-family depends on the browser Especifica la familia de fuentes para el texto (ej. Arial, Verdana).
font-size medium Define el tamaño de la fuente del texto.
font-weight normal Controla el grosor de la fuente (ej. normal, bold).
font-style normal Define el estilo de la fuente, como normal, italic, o oblique.
font-variant normal Transforma el texto en variantes como small-caps.
font Shorthand Propiedad abreviada para establecer font-style, font-variant, font-weight, font-size, line-height y font-family.
text-align start (depende del idioma) Alinea el texto dentro de un elemento (ej. left, right, center, justify).
line-height normal Controla la altura de cada línea de texto.
text-decoration none Aplica decoraciones al texto como underline, overline, line-through.
text-transform none Controla la capitalización del texto (ej. uppercase, lowercase, capitalize).
vertical-align baseline Alinea un elemento en relación con su línea base (ej. top, middle, bottom).
text-indent 0 Define la sangría de la primera línea de un bloque de texto.
letter-spacing normal Ajusta el espacio entre caracteres de un texto.
word-spacing normal Controla el espacio entre palabras.
white-space normal Determina cómo se manejan los espacios en blanco dentro de un elemento (ej. pre, nowrap, pre-line).

Característica Propiedades Descripción
Tamaño, Color y Decoración color Establece el color del texto de un elemento.
font-size Define el tamaño de la fuente del texto.
text-decoration Aplica decoraciones al texto como underline, line-through, o overline.
background-color Define el color de fondo de un elemento, afectando su área de contorno.
Decoración Personalizada text-shadow Añade una sombra al texto con opciones para definir la posición, el desenfoque y el color.
border Agrega un borde alrededor del elemento, con opciones para definir el estilo, el grosor y el color.
background-image Aplica una imagen de fondo a un elemento, lo que permite personalizar la apariencia visual de un área.
Característica Propiedades Descripción
Anchura y Altura width Define el ancho de un elemento, estableciendo su tamaño horizontal.
height Establece la altura de un elemento, controlando su tamaño vertical.
Eliminar el Borde border: none Quita el borde de un elemento, dejándolo sin contorno visible.
Propiedad Valor Inicial Descripción
list-style-type disc Define el tipo de marcador para los elementos de la lista (ej. circle, square, none).
list-style-position outside Establece la posición de los marcadores en relación con el contenido de la lista (inside o outside).
list-style-image none Permite utilizar una imagen como marcador de lista en lugar de un símbolo estándar.
list-style Shorthand Propiedad abreviada para definir list-style-type, list-style-position y list-style-image en una sola línea.
Propiedad Se Aplica a Valor Inicial Descripción
border-collapse Elementos table separate Determina si las celdas de la tabla tienen bordes separados (separate) o combinados (collapse).
border-spacing Elementos table cuando border-collapse: separate 0 Establece la distancia entre los bordes de las celdas de la tabla cuando los bordes están separados.
empty-cells Elementos table y td show Controla la visibilidad de las celdas vacías en una tabla (show para mostrar, hide para ocultar).
caption-side Elementos table-caption top Especifica la posición de la leyenda de una tabla (ej. top, bottom).
